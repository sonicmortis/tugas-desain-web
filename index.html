Progressive Web App (PWA)

Website ini sekarang sudah menjadi contoh Progressive Web App (PWA) yang mana berarti kita dapat menginstalnya ke layar utama (home screen) perangkat dan menganggapnya sebagai aplikasi. PWA memiliki 3 komponen utama, yaitu:

  1. File Manifest: File JSON yang berisi metadata mengenai aplikasi web seperti nama, nama pendek, ikon, warna tema, orientasi layar, dll.
  2. Service Worker: Script yang berjalan di background untuk mengelola caching, notifikasi push, fitur offline, dan intersepsi permintaan.
  3. HTTPS: PWA harus diakses melalui koneksi yang aman dan terenskripsi untuk melindungi data user.
